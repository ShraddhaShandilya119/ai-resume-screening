# AI Resume Screening System

## 📌 Description
This project analyzes resumes against job descriptions using AI and provides a relevance score with feedback.

## 🛠 Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- MongoDB
- Cohere AI (NLP)
- Firebase Authentication

## 🚀 Features
- Upload resume (PDF)
- Job description matching
- AI-based scoring
- User authentication
- Dashboard for results

## 📦 Setup
```bash
npm install
npm run dev
